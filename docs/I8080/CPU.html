    Overview

    Emulator for the Intel 8080 CPU.

    Load a ROM and execute it:

    cpu = I8080::CPU.new
cpu.load_file("path/to/rom")
cpu.run

    Alternatively, you can create a CPU in debug mode which has an embedded Disassembler that will print instructions as they are executed:

    cpu = I8080::CPU.new(debug: true)
cpu.load_file("path/to/rom")
cpu.step  # => 0000: 31 34 12  LXI    SP,$1234
cpu.step  # => 0003: 3E 56     MVI    A,$56

    This is intended to be used in applications that require an embedded Intel 8080 core.
